ZAD 1.1
Aplikacja Mapy z Leaflet


Technologie

Leaflet
Vite
npm

Funkcje

Podwójne kliknięcie na mapie dodaje marker.
Marker można przeciągać i aktualizować współrzędne w formularzu.
Formularz pozwala na edycję nazwy i opisu miejsca.
Koordynaty są automatycznie aktualizowane na podstawie lokalizacji markera.

Wymagania

Node.js
npm

Instalacja
pobrac repozytorium
cd zad1
npm install
npm run dev
