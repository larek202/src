ZAD 3.3
Formularz Rejestracji Użytkownika

Funkcjonalności
Walidacja pól formularza:
  -Imię: pole wymagane
  -Nazwisko: pole wymagane
  -Email: pole wymagane, musi mieć poprawny format
  -Hasło: pole wymagane, musi mieć co najmniej 6 znaków oraz zawierać przynajmniej jedną literę i jedną cyfrę
Komunikaty błędów wyświetlane obok odpowiednich pól
Potwierdzenie rejestracji po poprawnym wypełnieniu formularza

Wymagania

Node.js
npm


Instalacja
pobrac repozytorium
cd zad3
npm install
npm run dev
